HST library for C language.

About.

HST stands for "Http single threaded". This is a lightweight C library for writing HTTP application servers running in constrained environments such as embedded systems or low-end virtual private servers (VPS).

Servers written with HST are intended to be used with trusted clients or they must be put behind reverse-proxy such as Nginx since they are vulnerable to denial of service (DOS) attack known as slow requests.
